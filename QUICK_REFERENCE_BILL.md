## 🚀 How to Use

### 1. Create Order
```
Add products → Checkout → Pay
```

### 2. Admin Approves
```
Admin Dashboard → Orders → Click Approve
```

### 3. User Downloads Bill
```
Orders Page → Find order → Click "View Bill"
```

### 4. Print/Save
```
Ctrl+P (or Cmd+P) → Save as PDF
```
